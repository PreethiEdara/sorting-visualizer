## SORTING ALGORITHMS VISUALIZER
Developed a python application to visualize various sorting algorithms and understand them better.

**Implemented Algorithms:**  
 Bubble Sort, 
 Insertion Sort, 
 Selection Sort, 
 Merge Sort,
 Quick Sort,
 Heap Sort,
 Counting Sort
